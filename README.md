# Reactcal

A React resource calendar widget built with Vite and TypeScript, deployed to GitHub Pages.

![Resource Calendar Screenshot](https://github.com/user-attachments/assets/90a9ded6-2c34-4470-8f43-6c238a4dbacf)

## Features

- **Resource rows** – each row represents a resource (room, person, equipment)
- **2-hour time slot columns** – the timeline is divided into 2-hour blocks
- **Quarter-hour precision** – events can start and end at any 15-minute mark
- **Demo data** – 6 resources with 18 sample events across a full work day
- **Hover tooltips** – hover over any event to see its title and exact time range

## Live Demo

The app is deployed to GitHub Pages at `https://phi1010.github.io/Reactcal/`.
